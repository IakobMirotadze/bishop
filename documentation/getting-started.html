Getting Started

Prerequisites

- Node(12.22.8 or latter)

- pm2

- npm

- nodemon

Getting Started


1. First download the file and Unzip the downloaded file and folder structure you get

2. Open the folder nextly via Visial Studio Code

3. Edit config.js (your domain, your no-reply mail )

// /config.js file

 ....

 API_URL: "https://backend.nextly.shop",
 WEBSITE_URL: "https://nextly.shop",
 IMG_URL: "https://backend.nextly.shop",

 maillerConfig: {
 // host: 'smtp.gmail.com',
 // port: 465,
 // secure: true,
 // tls: { rejectUnauthorized: true },
 service: 'Gmail',
 auth: {
 user: 'noreply-mail@gmail.com',
 pass: 'password'
        }
    },
....

4. edit .env file in backend folder (mongod atlas or download mongodb enterprice) more details

//backend/.env file

ATLAS_URI=mongodb+srv://<user>:<password>@<cluster0.xmmg5.gcp.mongodb.net>/nextly
PASPORTJS_KEY=nextlykey

5. Open Visial Studio Code terminal

6. pm2 and nodemon instal global

sudo npm install -g pm2@latest nodemon

7. All npm package install

sudo npm run allnpmi

8. All Project start developer mode

sudo npm run allnpmdev

9. Add initial Data

Visit link: http://localhost:5000/installdb

10. Ready nextly

backend: http://localhost:5000

website: http://localhost:3000

admin: http://localhost:8000

11. All Next. build (for deployment)

sudo npm run allnpmbuild

12. All Next start and start backend (for deployment)

sudo npm run allnpmstart
