How It Works

We have used monorepo folder structure with Yarn Workspace. In our app we have three different services:

- backend(api)
- admin

- website

  • Ant Design
  • Tailwindcss

Shop Frontend

  • Next.js
  • Redux
  • Ant Design
  • Tailwindcss
  • LESS

Backend
